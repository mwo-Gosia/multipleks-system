# multipleks-system
Kod źródłowy programu Mulitipleks służący do zarządzania siecią kin. 

### Opis Projektu
System obsługuje dwa kina: Super Tarasy oraz Kino Galeria.   
Każde posiada oddzielny repertuar i sale.
